About Tab Plan

*New* Allow for user to pick randomly generated show in entire AniList database without signing in (requires no OAuth key). App will grab randomized entry via query request from AniList's GraphQL API and return a JSON response. Confirmed to work with JavaScript, but unsure if additional steps are necessary for TypeScript.

*MVP* Upon sign-in (from settings), user can choose show to watch/rewatch from their entire list (requires Implicit Grant OAuth key) via API.

*Next Step* Filter list by "Completed" (Rewatch) and "Plan To Watch" (Watch). Filtering by "Paused" and "Dropped" is a possible option, but filter by "Watching" is unnecessary.

*Optional Step* Implement Trakt API and sign-in for non-anime entries. Optional because everyone I know who rewatches non-anime shows just watch The Office or something. Low priority.
